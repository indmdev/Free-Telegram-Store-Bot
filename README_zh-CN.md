# 使用指南
1. 安装 Python 3.10
2. 安装任意版本的 git
3. 打开终端
4. 在终端中运行命令 "git clone https://github.com/indmdev/Free-Telegram-Store-Bot.git"
5. 在终端中运行命令 "cd Free-Telegram-Store-Bot-main"
6. 在终端中运行命令："pip install -r requirements.txt"
7. 在 https://ngrok.com 注册免费 NGROK 账号
8. 打开另一个终端并运行 Ngrok
9. 在 [@BotFather](https://t.me/Botfather) 创建您的新 Bot
10. 打开 config.env 文件
11. 添加您的 Bot Token（由 [@BotFather](https://t.me/Botfather) 提供）
12. 添加您的 Ngrok URL
13. 添加您的商店货币
14. 保存并关闭文件
16. 在终端中从 "Free-Telegram-Store-Bot-main" 文件夹运行 "python store_main.py" 命令
17. 完成
